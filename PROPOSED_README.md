## üöÄ Project Overview

This browser extension empowers developers and website owners with real-time insights into their web performance. By integrating directly with browser DevTools, it monitors and analyzes Core Web Vitals and other key performance metrics, providing actionable suggestions for optimization.

## üì¶ Installation & Setup

bash
# Clone the repository
git clone https://github.com/chirag127/WebPerf-Insight-Real-Time-Performance-Analyzer-Browser-Extension.git
cd WebPerf-Insight-Real-Time-Performance-Analyzer-Browser-Extension

# Install dependencies (using pnpm as an example)
pnpm install

# For development, run Vite's dev server (which WXT will leverage)
pnpm dev

# Load the extension in your browser (e.g., Chrome/Edge/Firefox)
# 1. Navigate to the browser's extension management page.
# 2. Enable 'Developer mode'.
# 3. Click 'Load unpacked' and select the 'dist' folder generated by Vite/WXT.
